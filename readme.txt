== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/lemiho-responsive-menu` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Product settings->Settings of plugin comfim something sytem of Lemiho Cart
